## xiaohongshuposter
创建小红书笔记的AI助手

# 演示
https://ai1.ai4uo.com (auth见代码）

# 功能
- 1.每天发布一条小红书笔记，主题为“历史上的今天”
- 2.每天发布一条小红书笔记，主题为“Github上有什么”
- 3.每天发布一条小红书笔记，主题为“今日读书摘录”
- 4.自动发布功能写了，但在主程序里没有使用
